Алгоритмика

Здесь (временно) живут конспекты занятий, которые я последние полтора года готовил для разных курсов и школ — в основном, для второго курса Tinkoff Generation.

Более-менее готово:

  • Дерево отрезков: на указателях, динамическое, пересистентное
  • Декартово дерево: явное, неявное, персистентое
  • Полиномиальное хэширование в задачах на строки
  • Центроидная декомпозиция
  • Heavy-light декомпозиция
  • Паросочетания: алгоритм Куна, покрытие DAG-а, лемма Холла
  • Дерево Фенвика: устройство, многомерное дерево, бинпоиск по дереву
  • Матроиды: алгоритм Радо-Эдмондса, расписания, паросочетания, линейная независимость
  • Ро-алгоритм Полларда
  • Sparse table
  • Поток минимальной стоимости: критерий оптимальности, потенциалы Джонсона
  • Битовое сжатие: битсет, нахождение цикла длины 3, перемножение матриц, метод Гаусса
  • Наименьший общий предок: свойства tin и tout, двоичные подъемы, сведение к RMQ, алгоритм Фараха-Колтона и Бендера
  • Классные задачки: пока что без решений

Дальше, в некоторых местах могут быть странно структурированные или незавершенные предложения. Это нормально.

Не очень готово:

  • Поиск строки в строке: префикс-функция, z-функция
  • Суффиксный массив
  • Геометрия: скалярное и векторное произведение, пересечение прямых, классы в C++
  • Оптимизации ДП: Кнут, разделяй-и-властвуй, Convex Hull Trick
  • Остовные деревья: алгоритм Прима и Крускала
  • Обратное по модулю
  • Метод отжига
  • Стресс-тестирование
  • Бор
  • Теория игр: эту статью определенно надо распилить на несколько
  • Выпуклые оболочки
  • Линейная алгебра: линейные операторы, матрицы, применения к динамике, метод Гаусса
  • CUDA: программирование на GPU (на английском)

Вообще не готово: * Алгоритм Карацубы: введение в разделяй-и-властвуй, мастер-теорема * Ахо-Корасик * Теорвер * Корневая декомпозиция * Теория информации * Функции потерь в машинном обучении * Память: кэширование, локальность, префетчинг, cache-oblivious алгоритмы (на английском) * Streaming SIMD Extensions: ассемблер, ускоряем программы в 8 раз (на английском)
